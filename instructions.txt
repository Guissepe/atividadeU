Git Basics Assignment - Your Instructions

1. Create a new folder and initialize the repository - V
2. Paste the "instructions.txt" file into this folder - V
3. Add a .txt file named "file-1" containing any text of your choice to the working directory - V
4. Create a second .txt file named "file-2" - V
5. Add "file-1" and "file-2" to the staging area - don't add "instructions.txt" - V
6. Change the initial text you added to "file-1" - V
7. Now add all working directory files to the staging area - V
8. Create the first commit - V
9. Create a second branch named "feature" (two commands are possible) - V
10. Add a third .txt file ("file-3.txt") to this branch - V
11. Create a new commit - V
12. Add the following text to "file-3": "I will be deleted"
13. Add the updated file to the staging area
14. Undo the staged change
15. Add the following text: "Please add me to the master/main branch"
16. Commit this latest change
17. Merge the "master" (or "main") branch with "feature"
18. Delete the "feature" branch

And most importantly: Have fun with the assignment :)
